## How To Use :closed_lock_with_key:

```bash
# Clone this repository
$ git clone git@github.com:mux-mux/kanban.git

# Go into the repository
$ cd kanban

# Install dependencies
$ npm install

# Open browser & listen to js changes
$ npm start
```
